1 Comentários em JavaScript


Comentários são linhas de código que o JavaScript irá ignorar intencionalmente. Os comentários são uma ótima
maneira de deixar anotações para você mesmo e para outras pessoas que mais tarde precisarão descobrir o que
aquele código faz.

Existem duas maneiras de escrever comentários em JavaScript:

Usar //instruirá o JavaScript a ignorar o restante do texto na linha atual. Este é um comentário in-line:



Você pode fazer um comentário de várias linhas começando /*e terminando com */. Este é um comentário de várias linhas:

/* This is a multi-line comment */

NOTA: Ao escrever o código, você deve adicionar comentários regularmente para esclarecer a função de partes do
aquele código faz.
seu código. Bons comentários podem ajudar a comunicar a intenção do seu código – tanto para
aquele código faz.
os outros quanto para você mesmo no futuro.
